# aws-SAM-Infrastructure-backup-api

## Introduction

This deploys a set of Lambda functions to automate backing up of AWS infrastructure with states. Eg: DynamoDB, RDS, Cognito, etc.

This uses AWS Serverless Architecture Model and Cloudformation to deploy APIS. The code is written in pure Python.

- AWS SAM
- Python
